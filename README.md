# 👨‍💻 Developer Journey

An interactive storytelling website that represents the journey of a developer — from beginner to professional.

---

## 🚀 Live Demo
🔗 https://developer-journey-tanuj.netlify.app/

---

## 🎯 Features

- 📖 Story-based UI (Chapter-wise flow)
- 🎨 Smooth animations using GSAP
- ⚡ Scroll-based transitions
- 🖱️ Interactive elements (buttons, counters)
- 📊 Progress bar
- 🎭 Clean and responsive design

---

## 🧠 Concept

This project focuses on transforming a simple webpage into an **immersive storytelling experience**.

Each section represents a stage:
1. Beginning
2. Struggle
3. Growth
4. Reality
5. Success

---

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
- GSAP (Animation Library)
